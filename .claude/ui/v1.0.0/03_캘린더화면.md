# 캘린더 화면 UI 분석

## 화면 개요
월별 달성률을 스티커(원형 컬러 도트)로 시각화하는 화면.
시작 날짜: 2026년 4월 1일 (이전 날짜 조회 불가).

## v1.0.0 스펙 결정사항

| 항목 | v1.0.0 동작 |
|------|------------|
| 날짜 탭 동작 | **없음** — 스티커 표시 전용, 별도 화면 없음 |
| 이전달 이동 | 2026년 4월이 최소값 → 그 이전 달 이동 불가 (좌측 화살표 비활성) |
| 다음달 이동 | 미래 달로 이동 가능 (우측 화살표 활성) |
| ? 버튼 동작 | 탭 시 스티커 범례 팝업 표시 (모바일: 호버 없이 탭으로 처리) |

---

## 레이아웃 구조
- 우상단: `?` 버튼 (스티커 안내 팝업 트리거)
- 헤더: 연도 + 월명 + 좌우 이동 화살표
- 범례 바: 4색 스티커 + 해당 월 집계 수
- 요일 헤더: SUN ~ SAT
- 날짜 그리드: 7열 × 최대 6행
- 최하단: BottomNavigationBar

## 필요한 위젯

### 레이아웃 위젯
| 위젯 | 용도 |
|------|------|
| `Scaffold` | 화면 기본 구조, `backgroundColor: Color(0xFFDEE0DF)` |
| `SafeArea` | 노치/홈바 안전 영역 처리 |
| `Column` | 전체 수직 레이아웃 |
| `Stack` | ? 버튼 우상단 고정 배치 |

### UI 컴포넌트
| 위젯 | 용도 |
|------|------|
| `Text` | 연도, 월명 텍스트 |
| `IconButton` | 좌/우 화살표 버튼 (색상: `#512DA8`) |
| `Row` + `Container(shape: circle)` + `Text` | 달성률 범례 바 (4색 × 집계 수) |
| 커스텀 `CalendarGrid` | 날짜 그리드 |
| `Stack` | 날짜 셀 내 숫자 + 스티커 도트 레이어 |
| `Container(decoration: BoxDecoration(shape: BoxShape.circle))` | 스티커 도트 |
| `Container(decoration: BoxDecoration(shape: BoxShape.circle, color: #512DA8))` | 현재 날짜 강조 배경 |
| `IconButton` 또는 `GestureDetector` | ? 버튼 |
| `AlertDialog` 또는 `OverlayEntry` | 스티커 범례 팝업 |
| `BottomNavigationBar` | 탭 네비게이션 |

### 스티커 색상 (project-overview.md 기준)
| 달성률 | 색상 | Hex |
|--------|------|-----|
| 0% | 표기 안 함 (스티커 없음) | - |
| 0% 초과 ~ 30% 미만 | 빨강 | `#e32910` |
| 30% 이상 ~ 60% 미만 | 노랑 | `#FFC943` |
| 60% 이상 ~ 100% 미만 | 초록 | `#13d62d` |
| 100% | 파랑 | `#46C8FF` |

### 달성률 범례 바
- `Row`로 구성: `[● 빨강 N] [● 노랑 N] [● 초록 N] [● 파랑 N]`
- 각 항목: 해당 색상 원형 + 해당 월에 해당 스티커가 붙은 날짜 수

### ? 버튼 팝업 (스티커 안내)
- 트리거: 탭 (모바일 환경, 호버 없음)
- 팝업 구성: 보라색 계열 배경 카드 + 스티커 색상별 달성률 범위 설명
- 구현: `showDialog(AlertDialog)` — 단순하고 관리 용이

### 날짜 그리드 셀 구성
```
Stack
 ├─ Container(circle, color: #512DA8)  ← 현재 날짜만
 ├─ Text(날짜 숫자)
 └─ Positioned(bottom) Container(circle, color: 스티커색)  ← 달성률 있는 날만
```

## 구현 계획

### 파일 위치
- `lib/features/calendar/view/calendar_screen.dart`
- `lib/shared/widgets/calendar_grid.dart`
- `lib/shared/widgets/achievement_sticker.dart`

### 구현 단계
1. `Scaffold` + `BottomNavigationBar` 기본 틀 구성
2. 월 네비게이션 헤더 구현
   - 연도/월 `Text` + 좌/우 `IconButton`
   - 현재 표시 월이 2026년 4월이면 좌측 버튼 비활성 (`onPressed: null`)
3. 달성률 범례 바 구현 (`Row` × 4)
4. 요일 헤더 `Row` (SUN~SAT, 텍스트 색상 `#512DA8`)
5. `CalendarGrid` 커스텀 위젯 구현
   - 해당 월의 1일 요일 오프셋 계산
   - `GridView` 7열로 날짜 배치
   - 각 셀: 날짜 숫자 + 스티커 도트 (달성률 → 색상 결정)
   - 현재 날짜 셀: 보라색 원형 배경
   - 날짜 탭: 인터랙션 없음 (`GestureDetector` 불필요)
6. ? 버튼: `Stack`으로 우상단 고정, 탭 시 `showDialog` 팝업
7. `CalendarViewModel`과 상태 연결 (월별 날짜별 달성률 데이터)

### 달성률 계산
```
달성률 = 실천완료된 todo 개수 / 전체 todo 개수
소수점 셋째자리에서 반올림 → 둘째자리까지 표기
```

### 월 이동 제한 로직
```
minMonth = DateTime(2026, 4)
currentDisplayMonth >= minMonth ? 좌측 버튼 활성 : 비활성
```
