# 로그인 화면 UI 분석

## 화면 개요
앱 최초 진입 시 표시되는 화면. 비회원 이용 불가이며 Google 소셜 로그인만 제공.

## 레이아웃 구조
- 상단 2/3: 앱 로고 이미지 영역
- 하단 1/3: 구글 소셜 로그인 버튼

---

## 필요한 위젯

### 레이아웃 위젯
| 위젯 | 용도 |
|------|------|
| `Scaffold` | 화면 기본 구조, `backgroundColor: Color(0xFFF3F4EB)` |
| `SafeArea` | 노치/홈바 안전 영역 처리 |
| `Column` | 로고(상단) + 버튼(하단) 수직 배치 |
| `Expanded` | 로고 이미지 영역 비율 확보 |
| `SizedBox` | 버튼 하단 여백 |

### UI 컴포넌트
| 위젯 | 용도 |
|------|------|
| `Image.asset` | 앱 로고 이미지 (`assets/images/just-three-logo.png`) |
| `SignInButton(Buttons.google)` | 구글 소셜 로그인 버튼 ([`sign_in_button`](https://pub.dev/packages/sign_in_button) 패키지) |

### Google 로그인 버튼
- 패키지: `sign_in_button: ^4.1.0`
- Google 공식 Sign-In 가이드라인을 패키지가 이미 준수
- `text: 'Google로 로그인'` 파라미터로 한글 텍스트 적용
- 별도 커스텀 래퍼 위젯 없이 `LoginScreen`에서 직접 사용

---

## 구현 계획

### 파일 위치
- `lib/features/auth/view/login_screen.dart`

### 구현 단계
1. `Scaffold` + 배경색(`#f3f4eb`) 설정
2. `Column`으로 로고(`Expanded`) + 버튼 레이아웃 구성
3. 로고 이미지 `assets/images/`에 추가 후 `Image.asset`으로 표시
4. `sign_in_button` 패키지 설치 후 `SignInButton(Buttons.google, text: 'Google로 로그인', ...)` 사용
5. 버튼 탭 → `AuthViewModel.signInWithGoogle()` 호출
6. 로그인 성공 → `go_router`로 `/todo` 라우트 이동
7. 로그인 실패 → `SnackBar`로 에러 메시지 표시

### 폰트 적용
- ThemeData에 fontFamily 전역 설정
- 한글: Noto Sans Korean / 영문: Google Sans
