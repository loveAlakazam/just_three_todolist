# 마이페이지 화면 UI 분석

## 화면 개요
사용자 프로필 확인, 프로필 편집, 회원탈퇴 기능 제공.
프로필 편집은 별도 화면으로 이동 (`go_router push`).

---

## My 탭 초기화면

### 레이아웃 구조
- 상단: 원형 프로필 이미지 + 사용자 이름 (가로 배치)
- 중단: 프로필 편집 버튼 (primary)
- 중단: 회원탈퇴 버튼 (gray)
- 최하단: BottomNavigationBar

### 필요한 위젯
| 위젯 | 용도 |
|------|------|
| `Scaffold` | 화면 기본 구조, `backgroundColor: Color(0xFFF3F4EB)` |
| `SafeArea` | 노치/홈바 안전 영역 처리 |
| `Column` | 전체 수직 레이아웃 |
| `Row` | 프로필 이미지 + 이름 가로 배치 |
| `CircleAvatar` | 원형 프로필 이미지 (기본 에셋 또는 Supabase Storage URL) |
| `Text` | `{사용자 이름} 님` (폰트 색상 `#512DA8`) |
| `ElevatedButton` | 프로필 편집 버튼 (`#512DA8` 배경) |
| `ElevatedButton` | 회원탈퇴 버튼 (gray 배경) |
| `AlertDialog` | 회원탈퇴 확인 팝업 |
| `BottomNavigationBar` | 탭 네비게이션 |

---

## 프로필 편집 화면

### 레이아웃 구조
- 상단: 원형 프로필 이미지 + 카메라 버튼 (이미지 우하단 오버레이)
- 중단: 이름 레이블 + 텍스트 입력 필드 (가로 배치)
- 중단: 수정하기 버튼 (primary)
- 최하단: BottomNavigationBar (My 탭 활성 상태 유지)

### 필요한 위젯
| 위젯 | 용도 |
|------|------|
| `Scaffold` | 별도 라우트 화면 기본 구조 |
| `SafeArea` | 노치/홈바 안전 영역 처리 |
| `Column` | 전체 수직 레이아웃 |
| `Stack` | 프로필 이미지 + 카메라 버튼 오버레이 |
| `CircleAvatar` | 원형 프로필 이미지 |
| `Positioned` + `GestureDetector` | 카메라 버튼 (이미지 우하단 고정) |
| `Container` + `Icon(Icons.camera_alt)` | 카메라 아이콘 버튼 |
| `showModalBottomSheet` | 이미지 업로드 / 이미지 제거 선택 팝업 |
| `ListTile` × 2 | 팝업 내 "이미지 업로드" / "이미지 제거" 항목 |
| `Row` | "이름" 레이블 + `TextField` 가로 배치 |
| `TextField` | 이름 입력 필드 (기존 이름 prefill) |
| `ElevatedButton` | 수정하기 버튼 (`#512DA8` 배경) |
| `BottomNavigationBar` | My 탭 활성 상태 유지 |

### 이미지 선택 BottomSheet (카메라 버튼 탭)
```
showModalBottomSheet
 ├─ ListTile "이미지 업로드"  → image_picker로 갤러리 접근 (권한 요청 포함)
 └─ ListTile "이미지 제거"   → 기본 이미지 에셋으로 임시 상태 변경
```

> **임시 적용 상태**: 이미지 업로드/제거는 수정하기 버튼 탭 시에만 최종 저장됨.
> 수정하기 없이 화면 이탈 시 변경 내용 미반영.

### 회원탈퇴 AlertDialog
```
AlertDialog
 ├─ title: "탈퇴하시겠습니까?"
 ├─ TextButton "취소"  → 팝업 닫기
 └─ TextButton "확인"  → ProfileViewModel.deleteAccount() 호출
```

## 구현 계획

### 파일 위치
- `lib/features/profile/view/my_screen.dart`
- `lib/features/profile/view/edit_profile_screen.dart`

### My 탭 구현 단계
1. `Scaffold` + 배경색 + `BottomNavigationBar` 기본 틀 구성
2. `Row(CircleAvatar + Text)` 프로필 섹션 구현
   - `CircleAvatar`: Supabase Storage URL 이미지 (없으면 기본 에셋)
3. 프로필 편집 버튼 탭 → `go_router.push('/profile/edit')`
4. 회원탈퇴 버튼 탭 → `showDialog(AlertDialog)` 표시
5. AlertDialog 확인 → `ProfileViewModel.deleteAccount()` 호출 → 로그인 화면으로 이동

### 프로필 편집 화면 구현 단계
1. `Scaffold` + 배경색 + `BottomNavigationBar`(My 활성) 기본 틀
2. `Stack`으로 `CircleAvatar` + `Positioned` 카메라 버튼 오버레이
3. 카메라 버튼 탭 → `showModalBottomSheet` 표시
   - "이미지 업로드": `image_picker`로 갤러리 접근 + 권한 요청 → 임시 `File` 상태 저장
   - "이미지 제거": 기본 이미지 에셋으로 임시 상태 변경
4. 이름 `TextField` 구현 (기존 이름으로 prefill)
5. 수정하기 버튼 탭 → `ProfileViewModel.updateProfile()` 호출
   - 이미지 업로드 케이스: Supabase Storage 업로드 → URL 저장
   - 이미지 제거 케이스: Storage 삭제 + `null` 처리
   - profiles 테이블 업데이트 → My 탭으로 `go_router.pop()`

### 이미지 처리 플로우
```
카메라 버튼 탭
 └─ "이미지 업로드" → image_picker → 임시 File 상태 (ViewModel)
 └─ "이미지 제거"  → 임시 null 상태 (ViewModel)

수정하기 버튼 탭
 └─ 업로드 케이스: Supabase Storage PUT → URL 획득 → profiles UPDATE
 └─ 제거 케이스:   Supabase Storage DELETE → profiles UPDATE (avatar_url = null)
```
