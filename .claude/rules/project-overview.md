# 프로젝트 개요

## 동기 및 배경

- Flutter 학습 목적의 1인 개발 프로젝트
- Claude Code를 활용하여 개발 경험을 키우는 것이 목표

## 서비스 개요

하루에 3개 목표계획을 실천하는 모바일 앱 플랫폼.

- 회원 전용 서비스 (비회원 이용 불가)
- 인증: Google 소셜 로그인으로 회원가입/로그인

## 주요 기능

### 1. Todo 탭

- todo list 구성: **목표**(텍스트, 한글 기준 최대 20자) + **달성여부**(boolean)
- 기본 3개 todo list 제공
- "목표추가하기" 버튼으로 새 todo list 추가 가능
- **최대 10개**까지 생성 가능

### 2. 캘린더 탭

- 날짜별 실천 달성률을 시각적으로 확인
- 시작날짜: 2026년 4월 1일 (이전 날짜는 조회 불가)

#### 달성률 계산

- `달성률 = (실천완료된 todo-list 개수) / (todo-list 개수)`
- 소수인 경우 소수점 둘째자리까지 표기
- 부동소수점 이슈 시 소수점 셋째자리에서 반올림

#### 달성률 UI (원형, 색상 구분)

| 달성률               | 색상        |
| -------------------- | ----------- |
| 0%                   | 표기 안 함  |
| 0% 초과 ~ 30% 미만   | `#e32910` |
| 30% 이상 ~ 60% 미만  | `#FFC943` |
| 60% 이상 ~ 100% 미만 | `#13d62d` |
| 100%                 | `#46C8FF` |

### 3. 마이 탭

#### 회원정보 표시

- 프로필이미지, 이름 표시

#### 프로필 편집

- "프로필 편집" 버튼 클릭 시 프로필 편집 화면으로 이동
- 수정 가능 항목: 프로필이미지, 이름
- 프로필 이미지 수정 옵션:
  - **사진업로드**: 모바일 기기 사진 접근권한 필요, 사진 1장 선택
  - **사진제거**: 기본이미지로 초기화

#### 회원탈퇴

- "회원탈퇴" 버튼 클릭 시 확인 팝업 표시 ("탈퇴하시겠습니까?")
- 팝업에서 재확인 후 탈퇴 처리
