## 1. Todo 기능 고도화

### 달성 목표 숨김/숨김해제 설정

- **배경**: v1.0.0에서는 달성 체크 후 숨김 처리 기능 보류. 잘못 눌렀을 때 되돌릴 수 없는 UX 문제가 있어 고도화로 미룸.
- **계획**: 마이페이지 내 설정 메뉴 추가
  - "달성한 목표 숨기기" ON/OFF 토글 제공
  - ON 상태: 달성된 목표를 todo 목록에서 숨김 처리
  - OFF 상태: 달성된 목표도 목록에 계속 표시 (v1.0.0 기본값)
- **관련 화면**: `04_마이페이지화면` 설정 섹션 추가 필요

## 3. 오프라인 지원 고도화

### 로컬 DB + 동기화 전략

- **로컬 DB**: `drift` 또는 `Hive` 도입 검토
- **동기화**: 변경 큐(Change Queue) 방식으로 온/오프라인 전환 시 자동 동기화
- **우선순위**: 소셜 로그인 확장 이후 검토
